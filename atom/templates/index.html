Data

Data is the underlying element of atom, mainly including the storage management of original data

Operator

The operator is the only important output of Atom. An operator is generated by one or more fixed data sets. An operator also represents a new feature reconstructed from the original data. In a word, operator is the core concept of Atom.

Service

Atom provides service function for each operator, making feature engineering have real-time computing function.
